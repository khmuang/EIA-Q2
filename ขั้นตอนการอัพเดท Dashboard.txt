  2. คู่มือการอัปเดตข้อมูลในครั้งถัดไป (Update Manual)

  เพื่อให้คุณทำงานต่อได้ง่ายในครั้งหน้า ผมได้สรุปขั้นตอนและคำสั่งที่จำเป็นไว้ให้ดังนี้ครับ:


  ขั้นตอนการอัปเดต (3 Steps):
   1. เตรียมไฟล์: นำไฟล์ Excel ชุดใหม่ทั้ง 8 ไฟล์มาวางทับไฟล์เดิมในโฟลเดอร์ EIA file (ตรวจสอบว่าชื่อไฟล์เหมือนเดิม)
   2. รันคำสั่งอัปเดต: เปิด Terminal ในโฟลเดอร์โปรเจกต์แล้วพิมพ์คำสั่ง:
   1     python "EIA file/update_dashboard_complete.py"
      (คำสั่งนี้จะคำนวณข้อมูลใหม่, ตัดแถวส่วนเกินออก, และอัปเดตวันที่/เวลาเป็นปัจจุบันให้ทันที)
   3. Sync ขึ้น GitHub: หากต้องการนำข้อมูลใหม่ขึ้นเว็บไซต์ทันที ให้พิมพ์คำสั่ง:


   1     git add index.html "EIA file/update_dashboard_complete.py"
   2     git commit -m "Update dashboard data: $(Get-Date -Format 'dd/MM/yyyy HH:mm')"
   3     git push origin main
